Breve descrição da ideia Para o usuario nao ficar muito tempo na fila e ter uma noçao de quanto tempo ele ira gastar até o local. colocamos a localizaçao em tempo real tambem. isso o usuario ira poupar muito tempo da sua vida.


Explicação das telas criadas
Criamos a tela para facilitar o tempo de espera do usuario para comer alguma coisa e colocamos o tempo de espera e o tempo para ele se locomover.


Fluxo principal do usuário 
1. Tela de Login 
2. Tela inicial com estabelecimentos próximos
3. Tela de detalhes do estabelecimento
4. Tela para entrar na fila
5. Tela de status da fila / tempo de espera
6. Tela de avaliação do local
